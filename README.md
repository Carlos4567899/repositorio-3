# Juego de Adivinanza en JavaScript

Un juego interactivo que reta al usuario a adivinar un número aleatorio entre 1 y 100.

Características
- El sistema genera un número secreto al azar.
- El jugador recibe pistas ("mayor" o "menor").
- Contador de intentos.
- Interfaz web con HTML, CSS y JS.

Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

Instrucciones de uso
1. Clona el repositorio.
2. Abre `index.html` en tu navegador.
3. Ingresa un número e intenta adivinar.

Funcionamiento del código
El programa usa `Math.random()` para generar el número secreto. Cada intento del jugador se evalúa con condicionales (`if/else`) y se actualiza el DOM con mensajes de "muy alto" o "muy bajo". El juego termina cuando el número es correcto.
